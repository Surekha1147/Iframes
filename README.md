# 🌐 HTML `<iframe>` Tag

## 📌 Introduction

The `<iframe>` (Inline Frame) tag is used to embed another webpage or external content inside the current HTML document.

It allows developers to display content such as:
- YouTube videos
- Google Maps
- External websites
- Online documents
- Payment gateways

The iframe creates a separate browsing context within a webpage.

---

# 🏷 Basic Syntax

```html
<iframe src="URL"></iframe>
